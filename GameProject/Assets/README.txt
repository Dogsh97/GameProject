!! 프로젝트 폴더 가이드

## Art
게임에 사용되는 모든 시각 에셋 저장.
FBX, 텍스처, 애니메이션 파일은 반드시 같은 폴더에 묶는다.

* Characters: 플레이어 및 NPC 모델
* Environment: 배경 및 맵 오브젝트
* Props: 상호작용 가능한 소품

## Audio
BGM, 효과음, 음성 파일 저장.
카테고리별로 하위 폴더 생성 가능.

## Materials
공용 머티리얼 저장.
특정 모델 전용 머티리얼은 해당 Art 폴더에 포함.

## Prefabs
씬에서 사용하는 모든 프리팹 저장.
역할 기준으로 분류한다.

## Scenes
게임 씬 파일 저장.
Test 폴더에는 실험용 씬만 배치.

## Scripts
게임 로직 코드 저장.

* Core: 기본 시스템
* Gameplay: 게임 플레이 로직
* Managers: 전역 관리 스크립트
* Systems: 공용 시스템

## Settings
렌더링 및 프로젝트 설정 파일 저장.

## UI
UI 관련 프리팹 및 리소스 저장.

## VFX
파티클, 쉐이더, 시각 효과 저장.



!! 규칙
1. 외부 에셋은 _ThirdParty에만 저장한다.
2. 테스트용 파일은 _Tests 폴더 사용.
